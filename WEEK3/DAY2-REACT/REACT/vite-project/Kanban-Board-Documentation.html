1. Project Overview

TaskBoard Pro is a modern, interactive Kanban board application built with vanilla JavaScript, HTML5, and CSS3. It provides a clean interface for managing tasks across three columns: To Do, In Progress, and Done.

Key Features

  • Add tasks via button click or Enter key
  • Delete tasks with confirmation
  • Drag and drop tasks between columns
  • Persistent storage using localStorage
  • Responsive design with animations
  • Real-time task counting
  • XSS protection with HTML escaping

4. addTask(column)

Purpose: Create and add new tasks to specified column
addTask(column) { const input = document.getElementById(`${column}-input`); const text = input.value.trim(); if (!text) return; // Guard clause const task = { id: this.taskIdCounter++, text: text, createdAt: new Date().toISOString(), }; this.tasks[column].push(task); input.value = ""; this.renderTasks(); this.saveTasks(); this.updateTaskCounts(); }

Process: Input validation ‚Üí Task creation ‚Üí Data update ‚Üí UI refresh ‚Üí Persistence

5. deleteTask(taskId, column)

Purpose: Remove tasks from the board
deleteTask(taskId, column) { this.tasks[column] = this.tasks[column].filter( (task) => task.id !== taskId ); this.renderTasks(); this.saveTasks(); this.updateTaskCounts(); }

How it works: Uses Array.filter() to create new array without deleted task, then updates UI.

6. renderTasks()

Purpose: Update the UI to reflect current task state
renderTasks() { ["todo", "doing", "done"].forEach((column) => { const container = document.getElementById(`${column}-tasks`); container.innerHTML = ""; this.tasks[column].forEach((task) => { const taskElement = this.createTaskElement(task, column); container.appendChild(taskElement); }); }); }

Strategy: Complete re-render approach - clears containers and rebuilds from data.

7. createTaskElement(task, column)

Purpose: Generate DOM elements for individual tasks
createTaskElement(task, column) { const taskDiv = document.createElement("div"); taskDiv.className = "task"; taskDiv.draggable = true; taskDiv.dataset.taskId = task.id; taskDiv.dataset.column = column; taskDiv.innerHTML = ` <div class="task-content"> <p>${this.escapeHtml(task.text)}</p> <span>${new Date(task.createdAt).toLocaleDateString()}</span> <button onclick="kanban.deleteTask(${task.id}, '${column}')"> Delete </button> </div> `; return taskDiv; }

Security: Uses escapeHtml() to prevent XSS attacks from user input.

8. setupDragAndDrop()

Purpose: Enable drag-and-drop functionality between columns
setupDragAndDrop() { // Container events for each column ["todo", "doing", "done"].forEach((column) => { const container = document.getElementById(`${column}-tasks`); container.addEventListener("dragover", (e) => { e.preventDefault(); container.classList.add("drag-over"); }); container.addEventListener("drop", (e) => { e.preventDefault(); this.handleDrop(e, column); }); }); // Global task events using event delegation document.addEventListener("dragstart", (e) => { if (e.target.classList.contains("task")) { this.draggedTask = { id: parseInt(e.target.dataset.taskId), column: e.target.dataset.column, }; } }); }

Pattern: Uses event delegation for dynamic content and provides visual feedback during drag operations.

9. handleDrop(e, targetColumn)

Purpose: Move tasks between columns via drag-and-drop
handleDrop(e, targetColumn) { if (!this.draggedTask) return; const { id, column: sourceColumn } = this.draggedTask; if (sourceColumn === targetColumn) return; // Find and move the task const taskIndex = this.tasks[sourceColumn].findIndex( (task) => task.id === id ); const task = this.tasks[sourceColumn].splice(taskIndex, 1)[0]; this.tasks[targetColumn].push(task); this.renderTasks(); this.saveTasks(); this.updateTaskCounts(); }

Logic: Validation ‚Üí Find task ‚Üí Remove from source ‚Üí Add to target ‚Üí Update UI

‚ö†Ô∏è Critical Implementation Details

  • State management: this.draggedTask maintains drag state throughout the operation

updateTaskCounts() { document.getElementById("todo-count").textContent = this.tasks.todo.length; document.getElementById("doing-count").textContent = this.tasks.doing.length; document.getElementById("done-count").textContent = this.tasks.done.length; }

11. saveTasks()

Purpose: Persist data to browser's localStorage
saveTasks() { localStorage.setItem("kanban-tasks", JSON.stringify(this.tasks)); localStorage.setItem("kanban-counter", this.taskIdCounter.toString()); }

Data: Saves both task data and ID counter for complete state preservation.

12. loadTasks()

Purpose: Restore data from localStorage on page load
loadTasks() { const savedTasks = localStorage.getItem("kanban-tasks"); const savedCounter = localStorage.getItem("kanban-counter"); if (savedTasks) { this.tasks = JSON.parse(savedTasks); } if (savedCounter) { this.taskIdCounter = parseInt(savedCounter); } this.renderTasks(); }

Safety: Includes existence checks before parsing to prevent errors.

13. escapeHtml(text)

Purpose: Prevent XSS attacks by escaping HTML characters
escapeHtml(text) { const div = document.createElement("div"); div.textContent = text; return div.innerHTML; }

Security: Converts dangerous characters like <script> to safe HTML entities &lt;script&gt;.
